# Piggo
專題名稱：整合行程管理與社群分享的智慧旅遊應用系統  
設計理念：包含旅行前中後的多功能旅遊APP  
主要功能分成六個面向：  
．開始使用：點入APP後會有登入註冊頁面，登入後會進行旅行人格測驗，希望透過AI分析推薦適合每個使用者的行程或景點。  
．計畫：安排行程、建立行李清單、以及能在群組內討論跟投票。  
．首頁：包含天氣、群組分帳、景點推薦、交通、導航至下一站的景點、翻譯與匯率轉換、群組定位分享。  
．相機：提供AI拍照指導、景點相關照片。  
．我的：分享貼文或行程，用來經營個人旅遊社群。  
．交流：分為推薦版和交流版，推薦版是推薦別人的分享的景點或行程，交流版是讓大家可以對當地的旅遊資訊做提問和回答。
